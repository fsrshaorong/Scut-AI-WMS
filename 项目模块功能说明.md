## <a id="sec-state-machine"></a>十五、二维码状态机

### 15.1 状态定义

| 状态 | 含义 | 进入条件 |
|:---|:---|:---|
| 待入库 | 入库单已创建，等待收货 | 新建入库单 |
| 在库 | 已收货上架，可供使用 | 确认入库 / 扫码入库 |
| 待出库 | 已被出库单拣选，等待确认 | 新建出库单（FIFO） |
| 已出库 | 已确认出库 | 确认出库 / 扫码出库 |
| FROZEN | 已封存，暂停流转 | 封存操作 |

### 15.2 状态转移图

```
待入库 ──确认入库──→ 在库 ──FIFO拣选──→ 待出库 ──确认出库──→ 已出库
                       │                  │
                       │   封存           │   (不可封存)
                       ▼                  │
                    FROZEN                │
                       │                  │
                       │   解封           │
                       ▼                  ▼
                     在库               已出库
```

### 15.3 互斥锁矩阵

| 当前状态 | 扫码入库 | 扫码出库 | 封存 | 解封 |
|:---|:---|:---|:---|:---|
| 待入库 | ✅→在库 | ❌ | ❌ | ❌ |
| 在库 | ❌"已入库" | ❌"需先拣选" | ✅→FROZEN | ❌ |
| 待出库 | ❌"已被拣选" | ✅→已出库 | ❌ | ❌ |
| 已出库 | ❌ | ❌"已出库" | ❌ | ❌ |
| FROZEN | ❌"已封存" | ❌"已封存" | ❌"重复" | ✅→在库 |
